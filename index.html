Random Topic Picker

I developed a web application called "Random Topic Picker" for Writing Assessment Test(WAT) for MBA preparations. The entire web app is built on Express and node.js with a MongoDB database. As of now, you need to manually run the seeds to insert the topics in the database, but I'm working on it! I used Bootstrap for the styling and method-override for manipulating HTTP requests.

Arduino Based Multimeter

It is an integrated version of two types of meters. The AVO meter for current, voltage and resistance and LC meter is used to measure capacitance and resistance.IR remote is used to instruct which channel to execute. The output of each individual circuit is displayed on the LCD screen.

Find-A-Place

Fina-A-Place is a simple web service created using ExpressJS, Bootstrap, EJS and MongoDB Atlas. It is hosted on Heroku. Using Mapbox's APIs, it enables the user to find any place on the face of earth. You can also view all the last visited places in the form of a cluster map and can even see additional details about the place.
