Arrow OS

A command-line Operating System built in Visual Studios using Cosmos Development Kit. It consists of a File System and Calculator and can perform basic functionalities such as Reboot, display Date & Time, System Lock, and Softwares such as SnakeGame and Text Editor.

Arduino Based Multimeter

It is an integrated version of two types of meters. The AVO meter for current, voltage and resistance and LC meter is used to measure capacitance and resistance.IR remote is used to instruct which channel to execute. The output of each individual circuit is displayed on the LCD screen.
